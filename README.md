# Data-Science-2---Do-Bias-Metrics-Agree-
“Do Bias Metrics Agree?” is a student project comparing IBM AIF360 and Microsoft Fairlearn on two datasets: German Credit, predicting loan applicants credit risk, and COMPAS, assessing recidivism risk across ethnic groups.

The Datasets used are the German Credit from: https://archive.ics.uci.edu/ml/machine-learning-databases/statlog/german/german.data

The COMPAS Dataset was extracted from the AIF360 framework itself.

The main difference between these two frameworks is that AIF360 takes binary inputs only whereas Fairlearn can take multiple inputs and compare them.
